# Home Lab Monitoring Stack

Monitoring stack for my Home Lab based on the [blog post](https://last9.io/blog/prometheus-with-docker-compose/) by Last9.

## Technologies

- Docker
- Prometheus
- Node Exporter
- cAdvisor
- Grafana
- Alertmanager
